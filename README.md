# 🏦 Addis Bank System

A Python-based interactive banking management system built using **Object-Oriented Programming (OOP)** principles. This project simulates real banking operations including account creation, deposits, withdrawals, transfers, and transaction management.

The project was developed as part of a 10-day software engineering exercise focusing on Python, OOP concepts, SOLID principles, design patterns, and data structures.

---

## 🚀 Features

### 👤 Account Management
- Create different account types:
  - Savings Account
  - Current Account
  - Basic Account
- Automatically manage account information
- Store and search accounts using Account Registry
- View account details and balances

### 💰 Banking Operations
- Deposit money
- Withdraw money
- Transfer money between accounts
- Check account balance
- View transaction history

### 📊 Advanced Features
- Top accounts leaderboard
- Banking statistics
- Savings account interest calculation
- Branch tree balance overview
- Transaction undo functionality

### 🧩 Software Engineering Concepts
- Object-Oriented Programming (OOP)
- Encapsulation
- Inheritance
- Polymorphism
- SOLID principles
- Factory Pattern
- Singleton Pattern
- Observer Pattern
- Data Structures:
  - Registry
  - Binary Search
  - Tree
  - Graph
